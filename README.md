# C++ Academy 🚀

C++ Academy — це сучасна інтерактивна платформа для вивчення мови програмування C++. Проєкт складається з клієнтської частини (React + Vite) та серверної частини (Node.js + Express) для компіляції та виконання коду у безпечному середовищі.

## ✨ Ключові можливості

- **Інтерактивне IDE**: Вбудований редактор коду з підсвіткою синтаксису (Monaco Editor).
- **Термінал виконання**: Код компілюється та виконується безпосередньо в браузері. Підтримується ввід даних (cin) та вивід результатів.
- **Гейміфікація**: Глобальне відстеження прогресу, бали досвіду (XP), система рівнів, серії днів та досягнення.
- **Швидка навігація**: Панель команд (Command Palette) дозволяє миттєво переміщуватися між сторінками сайту.
- **Теоретичні уроки та Практика**: Розділи для вивчення теорії та вільного розв'язання задач.
- **Темна/Світла тема**: Платформа підтримує повне перемикання кольорової схеми (через Command Palette).

---

## 🔍 Панель команд (Command Palette)

Command Palette — це потужний інструмент для швидкострільної навігації та управління платформою без використання миші. Її можна викликати у будь-який момент комбінацією `Ctrl + K`. 

**Вона дозволяє:**
- **Швидку навігацію**: Миттєвий перехід на основні розділи (Навчання, Практика, Тести, Проєкти, Профіль).
- **Розумний пошук**: Ви можете просто почати вводити назву потрібної сторінки, і панель миттєво відфільтрує результати.
- **Керування темою**: Можливість швидко перемикати зовнішній вигляд платформи між "Світлою" (Light) та "Темною" (Dark) темами.
- **Повну підтримку клавіатури**: Пересувайтеся списком команд за допомогою клавіш зі стрілками (`Up` / `Down`) та натискайте `Enter` для застосування.

## ⌨️ Гарячі клавіші та Шорткати (Комбінації клавіш)

Платформа підтримує ряд зручних комбінацій для швидкої взаємодії:

| Комбінація | Дія | Опис |
| :--- | :--- | :--- |
| **`Ctrl + K`**<br>*(або `Cmd + K` на Mac)* | **Панель команд сайту** | Відкриває зручне меню глобального пошуку по всьому сайту та доступ до перемикання тем. *Працює з будь-якої сторінки за умов будь-якої розкладки.* |
| **`F1`** | **Панель інструментів IDE** | Працює **всередині редактора коду**. Відкриває вбудовану панель редактора (як у VS Code) для форматування коду, пошуку, заміни та інших команд редагування. |
| **`Ctrl + Enter`**<br>*(або `Cmd + Enter`)* | **Запуск коду** | Швидко запускає компіляцію написаного коду у середовищі уроку або практичного модуля. |
| **`Escape`** | **Закрити** | Швидко закриває панель команд сайту, модальні вікна чи інші спливаючі елементи. |

## 💻 Модуль Терміналу та IDE

Інтегрований термінал на сторінках Уроків та Практики надає можливість не лише переглядати статичний результат, а й динамічно взаємодіяти з програмою.

- **Сучасний редактор**: Створено на базі потужного *Monaco Editor*, який використовується у VS Code.
- **Поле для вводу (cin)**: Термінал містить окреме текстове поле "Ввід даних (cin)". Дані, які ви туди вписуєте перед натисканням на "Запустити", напряму передаються до стандартного потоку вводу `std::cin` C++.
- **Безпека та Таймаути**: Бекенд компілює ваш код у захищеному тимчасовому середовищі із встановленими обмеженнями часу (зазвичай ~5 секунд на виконання). Якщо програма зависне або потрапить в нескінченний цикл — сервер безпечно її зупинить та сповістить вас про таймаут.

## 🛠 Інструкція по запуску проєкту

Проєкт є монорепозиторієм і складається з двох різних частин, які повинні бути запущені одночасно та паралельно: **backend** та **frontend**.

### 1. Запуск серверної частини (Backend)

Бекенд працює як API-сервер: приймає С++ код, компілює його через `g++` та повертає результати.

> **⚠️ Зверніть увагу:** Бекенд вимагає обов'язкової наявності встановленого **компілятора C++ (`g++`)** у вашій системі (наприклад, через MinGW для Windows або GCC для Linux). Компілятор має бути додано до змінних середовища (PATH). Без нього код не зможе виконуватися.

1. Увійдіть у папку бекенду в терміналі:
   ```bash
   cd backend
   ```
2. Встановіть необхідні Node-модулі та залежності:
   ```bash
   npm install
   ```
3. *(Опціонально)* Створіть файл `.env` на основі наявного `.env.example`, якщо бажаєте змінити порт чи інші конфігурації.
4. Запустіть розробницький сервер:
   ```bash
   npm run dev
   ```
   *Тепер API бекенду запущено і воно очікує підключення на порті (за умовчанням 3000).*

### 2. Запуск клієнтської частини (Frontend)

Фронтенд — це весь візуальний інтерфейс користувача, написаний на React з використанням Tailwind CSS для стилізації. Його необхідно запускати в окремому вікні/вкладці термінала.

1. Відкрийте **новий термінал** та увійдіть у папку фронтенду:
   ```bash
   cd frontend
   ```
2. Встановіть усі залежності Vite-проєкту:
   ```bash
   npm install
   ```
3. Запустіть локальний клієнт-сервер:
   ```bash
   npm run dev
   ```

Після того як Vite-сервер завантажиться, він відобразить адресу в терміналі (зазвичай `http://localhost:5173`). Відкрийте це посилання у своєму сучасному веббраузері — вітаємо, ви можете починати навчання на власній платформі!
